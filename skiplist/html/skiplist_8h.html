Подробное описание

skiplist.h - библиотека с реализацией структуры данных "Очередь с пропусками".

SkipList - вероятностная структура данных, основанная на нескольких параллельных отсортированных связных списках с эффективностью, сравнимой с двоичным деревом (порядка O(log n) среднее время для большинства операций)
